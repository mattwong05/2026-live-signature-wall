# Roadmap

## Current

- 提供局域网签名页与大屏页
- 支持签名轨迹持久化与实时回放
- 支持管理页设置大屏背景图
- 支持管理页生成签名页二维码与清空签名
- 支持管理页编辑大屏标题
- 支持管理页维护签名端宣誓文案列表
- 支持管理页导出透明背景签名图片压缩包
- 支持管理页实时刷新签名数量
- 支持管理页实时调整背景图显示强度
- 已禁用管理页与其配置接口缓存，避免后台统计展示旧值
- 支持管理页触发大屏结束签名收束爆发动画
- 结束签名动画已支持加速汇聚和更小的中心能量点表现
- 结束签名动画已支持延后生成的小中心光点和平滑白光冲击波
- 支持手机端点击签名框进入旋转 90 度的宽屏签名模式，按钮贴到画布单侧
- 支持手机端在签名前手动选择宣誓文案
- 宽屏签名框已改为按视口动态铺满可用高度
- 宽屏签名浮层已修正背景卡片包裹范围，避免旋转工作区溢出
- 结束签名动画已支持在汇聚过程中重播各个签名笔迹
- 支持 GitHub Actions 自动构建 Windows 文件夹版 `.exe`
- Windows 启动后自动打开管理页与大屏页
- Windows CI 测试已处理 `TestClient` 与 sqlite 连接释放，避免构建前测试阶段失败
- Windows `.exe` 已兼容无控制台启动时的 `uvicorn` 日志初始化
- 移动端签名页已补强中央 Toast 与庆祝反馈
- 支持 Windows `.exe` 打包

## Next

- 增加会场管理页，支持清空队列和历史背景
- 增加局域网接入说明与设备检查页

## Later

- 支持签名者昵称或祝福语
- 支持导出签名历史为图片或视频
- 支持多大屏同步播放
